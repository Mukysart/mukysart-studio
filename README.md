# Mukysart Studio

A production-ready graphics design web application built with Next.js and Tailwind CSS.

This project is a modern, state-driven graphics editor inspired by tools like Figma and Canva.

## Features

- **Component-based architecture** with React and Next.js.
- **State-driven UI** using React Context for global state management.
- **Rich toolset** including selection, text, image, and shape manipulation.
- **Advanced vector editing** capabilities with a node editor.
- **Project management** for saving, loading, and exporting designs.
- **Professionally designed UI** with ShadCN components and Tailwind CSS.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.
